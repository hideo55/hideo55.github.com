ArangoDB

  • AvocadoDBから改名
  • Schema Free

  • シェルはJavaScript
  • REST API
  • AQL(ArangoDB Query Language)

ArangoDB with Perl

my $db = ArangoDB->new({ 
    host => 'localhost' 
});
my $coll = $db->create('foo');
my $doc = $coll->save({ 
    bar => 'baz', qux => 'quxx' 
});
my $docs = $coll->first_by_example({ 
    bar => 'baz' 
});
